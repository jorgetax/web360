# Tienda en línea

Este proyecto es un sistema de ventas en línea, donde los usuarios pueden registrarse, iniciar sesión, ver productos,
agregar productos al carrito, realizar pedidos y ver el historial de pedidos.

## Tecnologías

- Node.js
- Express
- React
- SQL Server
